Get Started

In this repository you can find example of unidirection web application builded with React, Redux and Akka. Project done on Scala and ES6. Communication between client and server working on top of websocket connection with JSON API. To do some changes on server, client must put command in channel, server will handle it and produce event that will be published across connected clients.

For more information please read README at authors Github: https://github.com/ArchDev/unidirectional-akka-redux/blob/master/README.md
